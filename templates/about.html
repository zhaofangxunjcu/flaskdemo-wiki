About This Project

This project was built with Flask and Wikipedia's API as part of the CP1404 programming course.

It's a simple web app that allows users to search for any term and receive a summary directly from Wikipedia.

Features include search, disambiguation handling, and dynamic page titles.

Created with ❤️ by a JCU student.
